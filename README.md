# # Financial Market Forecasting – Systematic, Bibliometric, & Content Analysis (JRFM Manuscript: jrfm-4129983)

## Supplementary Materials

This repository contains the bibliometric analysis outputs generated using the Bibliometrix/Biblioshiny framework for the study:

Wafi, A.S., El-Halaby, S., Ahmed, H. (2026). Financial-market forecasting and modelling from econometrics to AI: An integrated systematic and bibliometric review with content synthesis (1990–2024). Journal of Risk and Financial Management.

The bibliometric analysis was conducted using the bibliometrix package (version 4.2.0) through its Biblioshiny interface, implemented in R (R Foundation for Statistical Computing, Vienna, Austria) and executed within RStudio (Posit PBC, Boston, MA, USA).**.

---

## Repository Structure

The repository contains the outputs generated during the bibliometric analysis:

**1. Overview**  
General bibliometric statistics, annual scientific production, citation trends, and three-field plots.

**2. Sources**  
Analysis of the most productive journals and sources, including source impact and production over time.

**3. Authors**  
Information on author productivity, influence, affiliations, and country distribution.

**4. Documents**  
Document-level analyses including global and local citation scores, keyword frequency, and topic trends.

**5. Clustering**  
Results of bibliographic coupling used to identify clusters of related research.

**6. Conceptual Structure**  
Conceptual mapping of the field including network analysis and factorial analysis.

**7. Intellectual Structure**  
Co-citation network analysis and historiographic mapping of influential publications.

**8. Social Structure**  
Collaboration networks and world maps illustrating international research cooperation.
